# 🪄 وب‌سایت فروشگاه قلاب جادویی (Gholab Jadooi)

این مخزن (Repository) مربوط به کدهای فرانت‌اند وب‌سایت فروشگاهی «قلاب جادویی» است.
برای مشاهده نقشه راه، تکنولوژی‌های استفاده شده و جزئیات معماری دیتابیس، لطفاً به فایل [STRATEGY.md](./STRATEGY.md) مراجعه کنید.
